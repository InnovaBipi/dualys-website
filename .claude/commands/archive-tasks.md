# archive-tasks (DEPRECATED)

This command has been deprecated. Tasks are now managed in the Asana Sprint Board.

Completed tasks are automatically archived by Asana when marked as Done.

## Use instead

- `/suggest-task` - See available [DUALYS] tasks
- `/update-task N done` - Complete a task
- `/develop` - Pick and execute the next task
