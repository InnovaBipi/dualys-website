# Dualys Website Content Structure

## Sitemap

```
/                           # Homepage
/about                      # About Dualys
/about/team                 # Leadership & Team
/about/partners             # Partner Ecosystem
/capabilities               # Our Capabilities (overview)
/capabilities/defense       # Defense Technologies
/capabilities/cybersecurity # Cybersecurity Solutions
/capabilities/biosecurity   # Biosecurity Capabilities
/capabilities/dual-use      # Dual-Use Technologies
/sectors                    # Sectors We Serve
/news                       # News & Insights
/news/[slug]                # Individual news articles
/contact                    # Contact Us
/legal/privacy              # Privacy Policy
/legal/terms                # Terms of Use
/legal/cookies              # Cookie Policy
```

## Page Purposes

### Homepage
- Hero with value proposition
- Key capability areas (4 pillars)
- Why Dualys (differentiators)
- Latest news/insights
- CTA to contact

### About Section
- Company story and mission
- European context and momentum
- Team profiles
- Partner ecosystem visualization

### Capabilities Section
- Each capability page:
  - Overview and applications
  - Key technologies
  - Use cases (civil + defense)
  - Related news/case studies

### News Section
- Filterable by category
- Search functionality
- Pagination
- RSS feed

### Contact
- Contact form
- Office locations
- Map integration
- Social links
